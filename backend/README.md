### Como rodar a aplicação no backend
- Acesse a pasta booksave > backend e rode o comando ``npm run start``

#### Principais bibliotecas utilizadas no backend
- Npm => v8.10.0
- Node => v16.13.1
- Express => v4.18.2
- Http => v0.0.1-security
- MD5 => v2.3.0
- Mongoose => v6.7.1
- Body-Parser => v1.20.1
- Cors => v2.8.5
